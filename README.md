# ContractShield AI

AI-powered contract risk analysis copilot for employment and business contracts.

## Features
- 📄 PDF/DOCX contract upload
- 🔍 Clause extraction and analysis
- ⚖️ Risk scoring (Legal, Financial, Operational)
- 📋 Negotiation checklist generation
- 📊 Risk dashboard visualization

## Quick Start

```bash
pip install -r requirements.txt
streamlit run contractshield_frontend.py
```

## Deployment

Deployed on Railway: [Live Demo](https://contractshield-ai-production.up.railway.app)
